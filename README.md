Lung CT Segmentation using U-Net

Overview

This project focuses on automated lung CT image segmentation using deep learning techniques based on the U-Net architecture. The model is trained to identify and segment lung regions from CT scans for medical image analysis.

Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Matplotlib

Features

- CT image preprocessing
- Lung region segmentation
- Model training and evaluation
- Visualization of segmentation results

Applications

- Medical image analysis
- Computer-aided diagnosis
- Healthcare AI systems
